
# FD-Recommend

> A Vue.js project

## Build Setup

``` bash
# 进入项目目录
cd /xxx/xxx/FD-Recommend

# 修改后台ip地址
/src/config/config.js (host)

# 安装依赖
npm install --save

# 运行
npm run dev

# 打包
npm run build

```
